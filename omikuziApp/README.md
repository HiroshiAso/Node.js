# おみくじアプリ (Omikuzi App)

## 概要
Node.js と Express で作成されたシンプルなおみくじアプリです。
アクセスするたびに「大吉」～「凶」の運勢をランダムに表示します。

## 動作環境
- Node.js (v14以上推奨)

## セットアップと実行
1. 依存パッケージのインストール
   ```bash
   npm install
   ```
2. アプリケーションの起動
   ```bash
   npm start
   ```
3. ブラウザでの確認
   `http://localhost:3000` にアクセスしてください。
