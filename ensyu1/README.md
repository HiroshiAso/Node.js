# メッセージゲーム (ensyu1)

簡単なメッセージゲームの練習課題です。トップ画面からメッセージ表示とメンテナンス画面に遷移でき、メンテナンス画面で「いつ・どこで・誰が・何を」の各パーツを登録してランダムメッセージを生成します。

## 使い方

```powershell
cd c:\Users\tsuda_hi\Documents\github\Node.js\ensyu1
npm install
npm start
```

ブラウザで `http://localhost:3000` を開くとトップ画面が表示されます。

## 画面概要

- `/` トップ画面 (G01)
- `/message` メッセージ表示 (G02) — 各カテゴリからランダムに1件ずつ選び結合して表示
- `/mainte` メンテナンス (G03) — 登録済み項目の一覧と追加フォーム

追加フォームではカテゴリを選択してテキストを入力し、追加後はトップ画面へ戻ります。データは `data/*.json` に保存されるためサーバー再起動時も保持されます。
